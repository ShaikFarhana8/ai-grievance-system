# AI Grievance System (NYC 311 NLP Project)

## 📌 Overview
This project uses Natural Language Processing (NLP) and Machine Learning to automatically classify citizen grievance complaints into government departments using the NYC 311 dataset.

## 🧠 Features
- Text preprocessing (cleaning, tokenization)
- TF-IDF feature extraction
- Logistic Regression classification model
- Complaint type prediction

## 🛠 Tech Stack
Python, Pandas, Scikit-learn, NLTK, NLP

## 📊 Dataset
NYC 311 Service Requests Dataset (Kaggle)

## 🚀 Goal
To automate grievance routing and reduce manual workload in government complaint handling systems.
